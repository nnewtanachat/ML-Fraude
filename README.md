# Fraud Detection with Kafka and Machine Learning
Tech Stack
-Postgest
-Kakfa
-XGBoost

Conclusion
This project demonstrates how to build a real-time fraud detection system using Kafka for streaming data and XGBoost for machine learning. The Kafka consumer listens to incoming transactions, processes them through a pre-trained model, and stores the results in a PostgreSQL database. This architecture allows for scalable and efficient fraud detection in a production environment.
